# git_guard
**for Windows**
### 1.
 install Rust
### 2.
 clone repo
### 3. 
cd C:\users\%username%\Documents\GitHub\git_guard
### 4.
run `cargo run`

# git_guard extra

**for Linux and MacOS**
### 1.
install Rust
```bash
sudo apt install cargo
```
or
```bash
sudo dnf install cargo
```
### 2.
 clone repo
### 3.
 ```bash
 cd git_guard
 ```
### 4.
run `cargo run`
